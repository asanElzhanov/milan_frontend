# Brand Entity

## Purpose

Provides brand API methods, DTO adapters, query keys, and React Query hooks.

## Endpoints Used

- `GET /api/v1/catalog/brands/`
- `GET /api/v1/catalog/brands/{slug}/`

## Intentionally Not Included

- Brand listing pages.
- Brand landing pages.
- Mock brand data.

## Future UI

Brand UI will be implemented later inside catalog and product experiences.
