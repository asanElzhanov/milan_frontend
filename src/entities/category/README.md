# Category Entity

## Purpose

Provides category API methods, DTO adapters, query keys, and React Query hooks.

## Endpoints Used

- `GET /api/v1/catalog/categories/`
- `GET /api/v1/catalog/categories/{slug}/`
- `GET /api/v1/catalog/categories/tree/`

## Intentionally Not Included

- Catalog navigation UI beyond existing header widget usage.
- Catalog page filtering.
- Mock category data.

## Future UI

Category UI will be implemented as part of catalog pages and navigation refinements.
