# Product Entity

## Purpose

Provides product API methods, DTO adapters, query keys, and React Query hooks.

## Endpoints Used

- `GET /api/v1/catalog/products/`
- `GET /api/v1/catalog/products/{slug}/`
- `GET /api/v1/catalog/products/{slug}/similar/`

## Intentionally Not Included

- ProductCard and ProductGrid UI.
- Add-to-cart, wishlist, reviews, or checkout logic.
- Mock product data.

## Future UI

Product UI will be implemented later in the entity/widget UI layers.
